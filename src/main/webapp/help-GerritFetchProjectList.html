Defines when and how the project list is fetched from Gerrit.

This list is used for project auto completion in the project configuration.

Load Project List on Startup

If checked, Gerrit will be queried for the project list as early as possible.

If unchecked, the project list will not be fetched until the first duration period has passed. Note: Auto-completion in the project configuration will not be availible during this period.

Refresh project list interval

The interval between fetches of the project list. A value of 0 disables the fetching altogether, which also means no project name auto-completion will be available for users configuring Jenkins projects.
